# Activity Log

## How to use it

### Server

To start the server

#### Windows

Run the application ```Server.exe```.

#### Linux

```bash
> make
```

Default port of the server is ```5000```

### GUI

GUI is available. You can find more information on the repository [Jack-GVDL/dashboard](https://github.com/Jack-GVDL/dashboard).
